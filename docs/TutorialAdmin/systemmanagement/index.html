System Management

Template File Management

Documents can be uploaded into Conforma to be used in any template/workflow. This can include documents such as Carbone templates, which are used to generate a .pdf with information from a completed form.

In order to manage the template files:

  1. Select Template File Management from the Manage dropdown menu

  1. This opens the File Management Form. In this window, select the template that you would like to use your file in, or leave this blank if it is not associated with a template

  2. You will then be prompted to select whether to add a new file or update an existing file

  3. You can then choose to enter the subfolder to upload your file into

  4. Select Click to Upload to choose your file to upload. You can add a description of the file

  1. Select Review and Submit to confirm.

Uploading reference docs

Reference docs can be uploaded into Conforma to assist users with processes or regulations associated with workflows. For example, a regulatory authority may link to guidance documents associated with registering a medicine in its jurisdiction.

  1. Begin by selecting Manage Reference Docs under the Manage dropdown menu.

  1. On the Update Files page, you can select Click to Upload to select the reference document you wish to include in the system

  2. You can add a description of the document, which is what will appear in the menu. Select Next

  1. You can choose whether the reference document will be visible to internal or external users (or both). Reference documents will be available from the Help dropdown menu for external users, or the Reference dropdown menu when only visible to internal users

  1. Once happy with your upload and selection, click Review & Summary

  2. Review your uploads and then select Submit Application to proceed or Back to form to make any changes.

Removing reference docs

  1. To remove an existing reference doc, or to change who can see it, begin by selecting Manage Reference Docs under the Manage dropdown menu

  1. On the Update Files page, you can skip past the upload process by selecting Review & Summary

  2. On the review page, you can see documents that have previously been uploaded and are visible to external and/or internal users

  1. Select the blue edit icon to make changes.

  2. Documents visible to external users and internal users will be displayed. You can then check or uncheck which documents you would like available to each type of user

  1. Once happy with your changes, select Review & Summary

  2. Confirm you are happy with the amendments and select Submit Application. Your changes will be reflected for internal and external users.
